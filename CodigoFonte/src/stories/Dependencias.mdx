## Observações e comandos úteis

- Para instalar todas as dependências:

```bash
npm install
```

- Para rodar o Storybook (porta padrão 6006):

```bash
npm run storybook
```

- Para rodar a aplicação Next.js em modo dev:

```bash
npm run dev
```

- Se houver problemas com o Storybook relacionados a versões do Node.js, verifique a versão atual:

```bash
node --version
```

Recomendamos manter o Node.js atualizado (pelo menos v20.19+ ou v22.12+ para a versão atual do Storybook no projeto).
